¿Qué es este Dashboard?

Esta es una guía interactiva para conducir sesiones de TRSB. Te acompaña paso a paso a través de las 8 fases del protocolo, mostrándote qué hacer en cada momento y qué señales observar.

Cómo usarlo
Sigue las 8 fases
Navega con las flechas o toca cada fase en el menú lateral
Usa el cronómetro
Registra la duración de tu sesión con el timer en la barra superior
Registra las escalas
NEA (perturbacion) y NCI (creencia positiva) - toca para actualizar
Este dashboard es una guía de referencia. Adapta siempre el protocolo a las necesidades específicas de cada sesión.
